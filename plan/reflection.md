# 反思纲要
很多日常错误是重复的，要记得定时更新和删除，提炼最核心的点，和最有效的调整手段。

# 反思
1. 现在生活很累，最本质的原因是休息不好，自己给自己加压。我觉得我现在最重要的第一目标是保证休息。要让自己有全身心放松的时间。11点睡觉，7点起床。
2. 不要在意一时的得失，永远记住人生的路很长，这不是安慰自己。人生是一场很长的马拉松，我们都会以各种方式到达终点。我给自己的压力目前过于大了，我确实比起相同水平中最优秀的那一部分人，出现了差距，但是一定可以赶上去，甚至有机会超越。
3. 做好规划，做好每周每天的规划，养成习惯，减少时间的浪费
4. 用科学地自控手段去控制自己，努力让自己每一个希望完成的任务，阻力最小。
