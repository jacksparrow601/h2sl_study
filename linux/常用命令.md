
## 压缩解压缩
### tar
#### 解压文件
```shell
tar -xvf archive.tar
```
-x: 解压 tar 文件。
-v: 在终端显示详细信息。
-f: 指定要解压的 tar 文件的名称。
archive.tar: 要解压的 tar 文件的名称。

创建tar文件
```shell
tar -cvf archive.tar file1 file2 directory1

```
-c: 创建一个新的 tar 文件。
-v: 在终端显示详细信息。
-f: 指定创建的 tar 文件的名称。
archive.tar: 创建的 tar 文件的名称。
file1 file2 directory1: 要打包的文件和目录的列表。

-z 使用gzip压缩或者解压缩
-t 列出tar包的文件内容
